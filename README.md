## Freym-Boost-Bulk
Script para extraer los totales de un masivo de facturas. 

Versión -all para extraer el proveedor con el input cliente.

## Dependencias
Installar Python 3

pip install PyMuPDF pandas

## Ejecución
En ruta de facturas:

python extractor.py . -o salida.csv

python extractor-all.py .
